## API

|参数|说明|类型|默认值|
|--|--|--|--|
|type|类型，可选值为`polar`、`theta`| string |`theta`|
|animate|动画开关|boolean|true|
|colors|自定义颜色|string[]|--|
|height|指定图表的高度，单位为 'px'|number|400|
|data|图表的数据源|Array|[]|
|forceFit|图表的宽度自适应开关|boolean|true|
|radius|设置半径，[0-1]的小数|number(0-1)|1|
|innerRadius|内部极坐标系的半径|number(0-1)|0|
